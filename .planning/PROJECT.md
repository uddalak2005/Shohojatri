# Shohojatri Website Evolution

## Mission
A sanctuary for psychosocial welfare, bridging academic research with human empathy.

## Project Context
The project is being restructured from a single-folder React app to a monorepo-style structure with a Sanity CMS backend to enable dynamic content management.

## Requirements
- Support dynamic content for Home, Achintya, Gallery, Blog, and Contact pages.
- Strictly follow the existing frontend schema for backend data structures.
- Reorganize project into `frontend/` and `backend/` folders.

## Key Decisions
- **CMS**: Sanity CMS.
- **Project Structure**: Monorepo with `frontend` (React/Vite) and `backend` (Sanity).
